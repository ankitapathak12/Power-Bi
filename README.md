# Power-Bi
Power BI Sales Dashboard (2024) that highlights key metrics like profit, trends using DAX, data modeling &amp; interactive visuals. It also uses a Calendar Table for accurate time-based analysis, well-defined relationships to ensure efficient, reliable insights. Highlights performance across categories, regions, salespersons for better decision-making.


🔧 Tools & Techniques Used
•	Calendar Table (DAX): Used for managing and analyzing date-based data
•	Data Modeling: Established relationships for accurate reporting
•	DAX Measures: Created key KPIs like Sales, Profit, Quantity, and Margin
•	Interactive Visualizations: Designed for clear and impactful storytelling

📈 Key Insights
•	Electronics is the top-performing category
•	Asia contributes the highest sales
•	Peak sales observed during mid-year
•	Charlie is the top-performing salesperson
•	Overall profit margin is ~17.70%

•	Hands-on experience with DAX
•	Enhanced Data Visualization & Storytelling
